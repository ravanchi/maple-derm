UI Component System

A showcase of reusable UI components for better maintainability and consistency

Component Management System

The component system uses a central registry to manage all UI components, making it easy to initialize and use them across the site.

// Component Manager const ComponentManager = { components: {}, // Register a new component register: function(name, component) { this.components[name] = component; return this; }, // Initialize all registered components init: function() { Object.values(this.components).forEach(component => { if (typeof component.init === 'function') { component.init(); } }); } };

Header Component

The header component is automatically injected into the page at the #header-placeholder element.

// Usage const headerPlaceholder = document.getElementById('header-placeholder'); // The component is automatically initialized on page load // ComponentManager.components.header.init();

Footer Component

The footer component is automatically injected into the page at the #footer-placeholder element.

// Usage const footerPlaceholder = document.getElementById('footer-placeholder'); // The component is automatically initialized on page load // ComponentManager.components.footer.init();

Hero Component

Create hero sections programmatically:

// Create a page hero ComponentManager.components.hero.createPageHero( document.getElementById('custom-hero'), 'Custom <em>Hero</em> Title', 'This hero was created programmatically using our component system.' );

Button Component

Create different types of buttons and links:

// Create a primary button const primaryButtonDiv = document.createElement('div'); buttonContainer.appendChild(primaryButtonDiv); ComponentManager.components.button.createButton( primaryButtonDiv, 'Primary Button', '#', true ); // Create a secondary button const secondaryButtonDiv = document.createElement('div'); buttonContainer.appendChild(secondaryButtonDiv); ComponentManager.components.button.createButton( secondaryButtonDiv, 'Secondary Button', '#', false ); // Create a text link const textLinkDiv = document.createElement('div'); buttonContainer.appendChild(textLinkDiv); ComponentManager.components.button.createTextLink( textLinkDiv, 'Text Link Example', '#' );

Extending the Component System

You can easily create and register new components:

// Create a new component const ExampleNewComponent = { init: function() { console.log('Example new component initialized'); }, createExample: function(container, title, content) { if (!container) return; const exampleHTML = ` <div class="example-component"> <h3>${title}</h3> <div class="example-content"> ${content} </div> </div> `; container.innerHTML = exampleHTML; } }; // Register the new component ComponentManager.register('example', ExampleNewComponent);
